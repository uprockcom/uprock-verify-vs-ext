# Change Log

All notable changes to the "UpRock Verify" extension will be documented in this file.

## [1.0.0] - 2024-12-24

### Added

- Initial release
- Global verification across 6 continents (NA, EU, AS, AF, OC, SA)
- Quick dev verification (single region)
- Batch URL verification (up to 10 URLs)
- Verify URL from text selection or cursor position
- Core Web Vitals display (LCP, CLS, TTFB, FCP, TTI)
- Reachability and Usability scoring
- Results panel with detailed breakdown
- Account status and usage tracking
- Recent scans history
- Secure API key storage
- Status bar integration
- Right-click context menu for URLs
- Configurable settings
- Keyboard shortcuts
